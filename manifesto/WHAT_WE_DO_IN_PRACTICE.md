# What We Do In Practice
*Powered by EruditeWBT*

The manifesto is the philosophy. This is the daily behavior.

## Weekly
- Ship one proof artifact (demo/diagram/write-up).
- Ask one question.
- Help one person.

## Monthly
- Upgrade one system (not “learn a topic”).
- Publish one public explanation (blog/video/notes).

## Always
- Prefer building over consuming.
- Prefer clarity over jargon.
- Prefer systems over fragments.

## Proof Standard
If you claim a skill, you should be able to show:
- a project
- a diagram
- a short explanation
