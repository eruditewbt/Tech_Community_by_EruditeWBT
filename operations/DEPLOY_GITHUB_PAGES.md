# Deploy GitHub Pages (Docs Folder)

This repo includes a static website in `docs/` (landing page + civilization graph viewer).

## Enable Pages
1. GitHub repo → **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`
4. Folder: `/docs`
5. Save

## Expected URL
`https://eruditewbt.github.io/Tech_Community_by_EruditeWBT/`

## Notes
- The graph viewer reads `docs/assets/graph/graph.json` (web export).
- Full exports are in `civilization_graph/exports/`.
