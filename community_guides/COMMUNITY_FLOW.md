# Community Flow
*Powered by EruditeWBT*

This is how the ecosystem is designed to work:

1. **Explore** (graph + guides)
2. **Choose** (a path and a 90-day goal)
3. **Build** (weekly sprints + projects)
4. **Document** (proof of work + diagrams + writeups)
5. **Teach forward** (help someone behind you)
6. **Earn** (micro-projects → consulting → products)

Key formula:

`Field → Role → Skills → Tools → Projects → Income`
