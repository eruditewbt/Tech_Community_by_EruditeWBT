# How To Use The Civilization Graph
*Powered by EruditeWBT*

The Civilization Graph is a **map of work**.

It’s not meant to be “pretty visualization” — it’s meant to answer:

> “What should I learn next to become valuable?”

## What You’re Looking At
- **Nodes**: occupations, tools, technologies, industries, and computing pillars
- **Edges**: relationships (uses, related to, works in, digitized by, etc.)

## The Path Formula (Translation Layer)
Use this everywhere:

`Field → Role → Skills → Tools → Projects → Income`

## How To Navigate (Practical Steps)
1. Search your current role (or a role you want).
2. Pin it (click) to view its neighborhood.
3. Note connected:
   - technologies/tools
   - industries
   - related occupations
   - computing pillars (AI, data, cloud, security, etc.)
4. Choose one pillar to focus on for 30–90 days.
5. Pick a project track that matches that pillar.

## Example (Generic)
If a role connects to:
- `Data & Analytics`
- database tools/tech

Then your next step is not “learn everything”:
- Learn the minimum skills/tools to build one data project.
- Ship one portfolio artifact.
