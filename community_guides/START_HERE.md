# Start Here
*Powered by EruditeWBT*

This community is built around two things:

- A **map** of the modern world (the Civilization Graph)
- A **path** to grow (the Engineering Tree + projects)

If you feel “behind”, “stuck”, or overwhelmed by tech: you’re the target audience.

## The 5-Minute Flow
1. Open the guided entry point: `docs/START.html`
2. Explore the graph: `docs/graph.html`
3. Pick one path: `community_guides/CHOOSE_YOUR_PATH.md`
4. Choose one starter project: `projects/tracks/beginner_projects.md`
5. Join the weekly rhythm: `community_guides/WEEKLY_SYSTEM.md`

## The Path Formula
Use this to translate “information” into outcomes:

`Field → Role → Skills → Tools → Projects → Income`

## What We Do (In Plain Terms)
- We learn by **building** (not collecting courses).
- We document what we learn so knowledge compounds.
- We help each other turn skills into real opportunities.
