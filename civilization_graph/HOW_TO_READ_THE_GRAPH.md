# How To Read The Graph
*Powered by EruditeWBT*

Open the viewer:
- `docs/graph.html`

## Nodes
- **Occupation**: a real job role (from O*NET)
- **Industry**: where the role tends to exist
- **Technology/Tool**: what is used in the work
- **Computing pillar**: the digital backbone (data, AI, cloud, security, networks, automation)

## Edges (Relationships)
- `uses_tool`, `uses_technology`
- `works_in_industry`
- `related_occupation`
- `digitized_by` (role’s “digital gravity” toward computing pillars)

## The Translation Layer (Most Important)
`Field → Role → Skills → Tools → Projects → Income`

The graph answers the first half:
- field/role/tools

The project system answers the second half:
- projects → proof → income
