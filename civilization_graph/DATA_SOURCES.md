# Data Sources
*Powered by EruditeWBT*

This repo’s civilization graph export is built primarily from **O*NET** occupation tables (titles, tools/technology skills, related occupations, and metadata).

## What’s Included In The Web Export
- Occupations
- Industries
- Tools + technologies
- Related occupations
- Computing pillars + linking

## What Can Be Added (Heavier Graph)
- Skills / knowledge / abilities importance
- Tasks / work activities

## Notes
- O*NET is a US-focused dataset. The long-term goal is to add crosswalks and international occupation classifications.
- Always verify career decisions with multiple sources; this is a navigation tool, not personal career advice.
