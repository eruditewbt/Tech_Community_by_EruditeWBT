# Civilization Graph Use Cases
*Powered by EruditeWBT*

## 1) “What should I learn next?”
Search your role → pin it → list the connected tools/tech → pick one pillar.

## 2) “What jobs are adjacent to mine?”
Use related occupations to find “nearby upgrades”.

## 3) “How is my field being digitized?”
Follow `digitized_by` edges into computing pillars, then look at tools/tech nodes.

## 4) “What projects prove I can do this?”
Translate from tools/tech into a project:
- build a dashboard
- automate a workflow
- build a small API
- build a simple pipeline

Then use:
- `projects/tracks/`
