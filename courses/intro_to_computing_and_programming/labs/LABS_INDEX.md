# Labs Index

These labs are designed to be short and repeatable.

## Foundations Labs

- Lab F1: File management drill (create/move/rename/search)
- Lab F2: Software install drill (download → install → verify version)
- Lab F3: Inspect system processes (what runs when you open an app)

## Networking Labs

- Lab N1: DNS + HTTP walkthrough (diagram + explanation)
- Lab N2: Troubleshooting checklist (simulate “site won’t load”)

## Programming Labs (Python)

- Lab P1: Hello world + input/output
- Lab P2: Variables + types + debugging errors
- Lab P3: Conditions + boolean logic
- Lab P4: Loops + patterns
- Lab P5: Functions + refactoring
- Lab P6: Files + JSON

## Under-the-Hood Labs

- Lab U1: Compare “hello world” across 3 languages
- Lab U2: Dependency install + documentation (one package)
