# Introduction to Computing & Programming

This is a complete foundations course designed for the community:

- Absolute beginners are supported (including non-tech backgrounds).
- The goal is not memorization. The goal is confidence + systems understanding + proof-of-work.
- Every unit ends with a deliverable (a diagram, a writeup, a lab, a tiny program, a mini-project).

## Outcomes (What Learners Can Do After)

- Explain what a computer is: input → process → output, and where CPU/RAM/storage fit.
- Use a computer intentionally: files, folders, installs, updates, basic troubleshooting.
- Understand “how code runs”: interpreter vs compiler, build/run loop, errors.
- Write and run basic programs, read code, and debug.
- Recognize major language families and pick a first language for their goal.
- Build 2–3 beginner projects and publish proof (README + screenshots + short demo).

## Recommended Delivery

- Duration: 24 weeks (core) or 36 weeks (extended).
- Schedule: 2 sessions/week (one theory+demo, one lab+build).
- Community loop: weekly sprint + share progress on Discord.
