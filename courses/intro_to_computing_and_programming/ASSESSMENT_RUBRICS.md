# Assessment Rubrics

Use these to review learner work consistently.

## Rubric 1: Concept Explanation (0–3)

0
- Cannot explain the concept.

1
- Explains parts, but missing the core idea or confuses terms.

2
- Explains the core idea correctly in simple language.

3
- Explains the idea and gives a correct example and a correct diagram.

## Rubric 2: Lab Output (0–3)

0
- No working output.

1
- Partially working or unclear.

2
- Working output with basic correctness.

3
- Working output plus a small improvement (edge cases, clarity, extra test).

## Rubric 3: Proof-of-Work Packaging (0–3)

0
- No documentation.

1
- Minimal notes, hard to reproduce.

2
- README exists with steps to run and expected output.

3
- README + screenshot/recording + limitations + next steps.

## “Graduate the Week” Rule

To pass a week:

- Rubric 1 ≥ 2
- Rubric 2 ≥ 2
- Rubric 3 ≥ 1

## Capstone Rubric (0–12)

Score each category 0–3:

- Problem statement clarity
- System design (diagram + decomposition)
- Working MVP
- Documentation + demo proof
